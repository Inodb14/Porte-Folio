Objectif du Projet

Développer un système d'analyse RH prédictif pour une entreprise technologique de 1,470 employés, permettant l'optimisation de la gestion des talents et la planification stratégique des ressources humaines.

Défis RH Critiques

  • Risque de turn-over : 8% approchent de la retraite (117 employés)
  • Stagnation promotionnelle : 95.1% non éligibles aux promotions
  • Déséquilibre départemental : Concentration R&D excessive
  • Satisfaction mitigée : Distribution égale satisfaction/insatisfaction
  • Manque de visibilité : Pas de données prédictives sur les talents

Méthodologie & Architecture

Phase 1 : Audit SIRH & Collecte de Données (3 semaines)

Extraction données employés, évaluations performance, historique promotions, géolocalisation. Intégration API SIRH avec validation qualité données et nettoyage.

Phase 2 : Modélisation Prédictive RH (4 semaines)

Développement algorithmes prédiction churn, scoring promotion, analyse satisfaction. Machine Learning avec Random Forest et régression logistique pour patterns comportementaux.

Phase 3 : Dashboard Power BI Avancé (3 semaines)

Création visualisations interactives, KPIs temps réel, filtres dynamiques par département/niveau. Implémentation mesures DAX complexes et drill-down multi-niveaux.

Phase 4 : Alertes & Automatisation (2 semaines)

Configuration alertes automatiques seuils critiques, notifications départs imminents, recommandations promotions. Pipeline ETL automatisé avec refresh quotidien.

Phase 5 : Formation & Déploiement (1 semaine)

Formation équipes RH sur utilisation dashboard, interprétation métriques, prise de décision data-driven. Documentation complète et plan de maintenance.
